KAMeL Kymograph Analysis with Machine/u> Learning


KAMeL automatically segments kymographs automatically
using various image processing and machine learning methods. Given a set of images, it requires some training with manually
annotated lines, and then calculates the average run length and speed of traced particles .

To use this code, we assume a basic working knowledge of MATLAB: strings,
cell arrays of strings, comments, using the command line, etc.
